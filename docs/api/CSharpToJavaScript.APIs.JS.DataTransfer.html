EffectAllowed

The DataTransfer.effectAllowed property specifies the
effect that is allowed for a drag operation. The copy operation is used to
indicate that the data being dragged will be copied from its present location to the
drop location. The move operation is used to indicate that the data being
dragged will be moved, and the link operation is used to indicate that some
form of relationship or connection will be created between the source and drop
locations.

event, thus effectAllowed may be used to determine

Types

The DataTransfer.types read-only property returns an
array of the drag data formats (as strings) that were set in
the event. The order of the formats is the same order as the data
included in the drag operation.

public string[] Types { get; }

Property Value

string[]

An array of the data formats used in the drag operation. Each format is a
string. If the drag operation included no data, this list
will be empty. If any files are included in the drag operation, then one of the types
will be the string Files.

Remarks

The formats are Unicode strings giving the type or format of the data, generally given
by a MIME type. Some values that are not MIME types are special-cased for legacy reasons
(for example &quot;text&quot;).

SetDragImage(Element, long, long)

When a drag occurs, a translucent image is generated from the drag target (the element
the event is fired at), and follows the mouse pointer during the
drag. This image is created automatically, so you do not need to create it yourself.
However, if a custom image is desired, the
DataTransfer.setDragImage() method can be used to set the
custom image to be used. The image will typically be an {{HTMLElement(&quot;img&quot;)}} element
but it can also be a {{HTMLElement(&quot;canvas&quot;)}} or any other visible element.

public Unsupported SetDragImage(Element image, long x, long y)

Unsupported

Remarks

The method&apos;s x and y coordinates define how the image should
appear relative to the mouse pointer. These coordinates define the offset into the image
where the mouse cursor should be. For instance, to display the image so that the pointer
is at its center, use values that are half the width and height of the image.

This method must be called in the event handler.
