CellSpacing

While you should instead use the CSS
{{cssxref("border-spacing")}} property, the obsolete HTMLTableElement
interface&apos;s cellSpacing property represents the spacing
around the individual {{HTMLElement(&quot;th&quot;)}} and {{HTMLElement(&quot;td&quot;)}} elements
representing a table&apos;s cells. Any two cells are separated by the sum of the
cellSpacing of each of the two cells.

public string CellSpacing { get; set; }

Property Value

string

A string which is either a number of pixels (such as
&quot;10&quot;) or a percentage value (like &quot;10%&quot;).

Remarks
