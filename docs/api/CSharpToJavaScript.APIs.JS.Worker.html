Worker(string, WorkerOptions)

The Worker interface of the Web Workers API represents a background task that can be created via script, which can send messages back to its creator.

public Worker(string scriptURL, WorkerOptions options)

Parameters

scriptURL string
options WorkerOptions

Remarks

Creating a worker is done by calling the Worker(&quot;path/to/worker/script&quot;) constructor.

Workers may themselves spawn new workers, as long as those workers are hosted at the same origin as the parent page. (Note: nested workers are not yet implemented in WebKit).

Not all interfaces and functions are available to scripts inside a Worker. Workers may use for network communication, but its responseXML and channel attributes are always null. (fetchhttps://developer.mozilla.org/en-US/docs/Web/API/Fetch_API is also available, with no such restrictions.)
